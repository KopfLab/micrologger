# micrologger

`micrologger` is a web-based graphical user interface for working with
[GEOM µLoggers](https://github.com/KopfLab/micrologger_device) in the lab. From
your browser you can organize measurements into **experiments**, **link [µLogger
devices](https://github.com/KopfLab/micrologger_device)** to them, 
**record and visualize** their time-series data, and **control
the devices** directly — all backed by a shared group database.

The app builds on [sddsParticle](https://github.com/KopfLab/sddsParticle) for the device
discovery and control.

The firmware and hardware of the µLogger devices this app controls are available
in the [micrologger_device](https://github.com/KopfLab/micrologger_device) repository.
